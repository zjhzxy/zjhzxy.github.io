Tensorflow 示例

Veröffentlicht am 2018-05-24

一个简单示例：

已知数据为：

x_data 为 100 个随机离散点；

y_data 数据为：为一条直线，斜率为 0.1，截距为 0.2；

数学模型为： y_data = 0.1 * x_data + 0.2

x_data = np.random.rand(100)

y_data = x_data * 0.1 + 0.2

现建立数学模型 y, k, b:

k 为随机数，初始值为 0.0；

b 为随机数，初始值为 0.0；

令 y = k * x_data + b;

现用已有的数据 x_data ， y_data 来训练模型 y = k * x_data + b 得出 k 和 b；

#首先：构造二次代价函数：

loss = tf.reduce_mean(tf.square(y_data -y))

#定义一个梯度下降法来进行训练的优化器：

optimizer = tf.train.GradientDescentOptimizer(0.1)

#最小化代价函数：

train = optimizer.minimize(loss)

完整代码：

x_data = np.random.rand(100)
y_data = x_data * 0.1 + 0.2

k = tf.Variable(0.)
b = tf.Variable(0.)
y = k * x_data + b

loss = tf.reduce_mean(tf.square(y_data-y))
optimizer = tf.train.GradientDescentOptimizer(0.1)
train = optimizer.minimize(loss)

init = tf.global_variables_initializer()

with tf.Session() as sess:
sess.run(init)
for step in range(201):
    sess.run(train)
    if (step%20==0):
        print(step, sess.run([k, b]))
